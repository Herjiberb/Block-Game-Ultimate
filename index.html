Frequently Asked Questions
How do I play a snapshot or older version?

Click the "Snapshots / Versions" button on the main screen. This will open a list. Click any version to launch it. Note: some versions are experimental and may be unstable.

My game is lagging. What can I do?

This is a browser-based game, so performance can vary. Try closing other browser tabs and applications. The "Current Development Build" often includes lag-reduction features, so make sure you're playing the latest version!

What is the "Geobotanical Combustion Thesis"?

That's the codename for the current version of the game. It's just a fun name! You can click it to see ideas for future updates.

Are there any secrets?

...Try typing 'hack' on the main launcher screen. Or, if you remember old video games, try the Konami code (Up, Up, Down, Down, Left, Right, Left, Right, B, A, Enter).

How do I change the launcher theme?

On the "Snapshots / Versions" list, try triple-clicking< (clicking three times fast) on a version name instead of single-clicking. This will set the launcher background to that version's banner. You can reset it from the Settings menu.
